# Hệ Thống Quản Lý Tài Sản

Ứng dụng giúp quản lý tài sản theo vai trò người dùng, bao gồm giao diện người dùng (frontend) và server (backend).

## Run Locally

### Start Backend

```bash
cd backend_qlts
npm install
npm run dev
```

### Start Frontend

```bash
cd frontend
npm install
npm run dev
```
